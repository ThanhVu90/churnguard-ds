# ChurnGuard DS CV Bullets

## English Version

- Built a customer churn prediction system using a reproducible Data Science pipeline with feature encoding, scaling, logistic regression, and threshold-based risk segmentation.
- Implemented logistic regression and core evaluation metrics from scratch in Python, including accuracy, precision, recall, F1, ROC AUC, and confusion matrix reporting.
- Designed an explainability layer that surfaces the top positive and negative feature contributions for each customer risk prediction.
- Structured the project as a production-style codebase with CLI commands, deterministic data generation, CSV export, and unit tests.

- Thiết kế lớp giải thích dự đoán, hiển thị các yếu tố làm tăng hoặc giảm churn risk cho từng khách hàng.
